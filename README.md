# Blackjack

This is a simple Blackjack game I created to test my Ruby skillz.

## Installation

Add this line to your application's Gemfile:

    gem 'blackjack'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install blackjack

## Usage

Gamble a little. All the fun, none of the risk :)
